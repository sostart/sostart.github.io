win下安装hexo并部署到github.io

  • 安装 msysGit
  • 安装 Node.js
  • 安装 npm
  • 安装 Hexo
  • 部署 GitHub
  • 编辑及发布

安装msysgit

官网 下载地址
本次安装下载的是便携版,下载解压即可(文件夹改名为msysGit)

安装nodejs

官网 下载地址

直接用安装版更好,可以跳过设置环境变量和安装npm了 = =

本次安装下载的是便携版,如下载到 D:\mysoft\Node.js,下载后将 D:\mysoft\Node.js 加入环境变量


安装npm

打开msysGit下的git-bash.bat

curl -L https://npmjs.com/install.sh | sh

安装hexo

打开msysGit下的git-bash.bat

npm install -g hexo

安装完成,下面是创建博客

创建博客文件夹(下面的博客目录都是指此文件夹),如在D:\mysoft下创建sostart.github.io文件夹

mkdir /d/mysoft/sostart.github.io

切换目录至刚创建的博客目录(后面的所有命令全部是在此目录下执行)

cd /d/mysoft/sostart.github.io

初始化博客

hexo init

安装依赖包

npm install

生成静态页面

hexo g

启动本地web服务(不要关掉窗口,关掉窗口web服务就停了)

hexo s

浏览器打开http://localhost:4000/,welcome to hexo
页面打开很慢的原因是默认主题使用了google cdn被墙的资源
可以替换成360提供的前端公共库cdn服务进行替换(或者直接更换成国产主题..)
进入博客目录将下面两个文件里的googleapis.com替换成useso.com
themes\landscape\layout_partial\after-footer.ejs
themes\landscape\layout_partial\head.ejs

部署到github

github注册就不介绍了教程很多

登陆github创建新的public仓库(New repository),仓库名为 你的账号.github.io
如我的是 sostart.github.io
编辑博客目录下的_config.yml,修改最后的deploy配置(sostart替换成你的账号)

打开msysGit下的git-bash.bat,目录切换至博客目录
生成静态文件

hexo g

部署

hexo d

等待部署完成,浏览器打开 你的账号.github.io 访问你的博客
(部署完后显示404的话耐心等待下,喝杯茶一会再看就出来了)

发布及部署

发布新的博文

打开msysGit下的git-bash.bat,目录切换至博客目录

hexo n "这是一篇新文章"

进入博客目录source_posts下,找到 这是一篇新文章.md,使用编辑软件编辑

markdown可以从这里学习
写完博文后,生成静态文件

hexo g

可以在本地预览http://localhost:4000/
然后发布到线上

hexo d
